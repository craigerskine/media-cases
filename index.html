Usage

Include the CSS:

<link rel="stylesheet" href="css/media.css" />

Sample Markup:
NOTE: replace ps4 with whatever case you want (see options below). Also, you can use your own CSS to style the overall layout, in the above demo I used Tailwind CSS.

<ul>
  <li><a href="#" class="media-case media-ps4"><img src="..." /></a></li>
  ...
</ul>

Options

Popular console classes: prefix with media-case media-
  • PS4 = ps4
  • PS3 = ps3
  • PS2 = dvd
  • PSone = cd
  • PS Vita = psv
  • PSP = psp
  • Switch = switch
  • Wii U = wiiu
  • Wii = wii
  • Gamecube = dvd
  • 3DS = n3ds
  • DS = nds
  • XBOX One = xbone
  • XBOX 360 = xb360
  • XBOX = xb
  • Dreamcast = cd
  • Sega CD/Saturn = cd-tl
  • TurboGrafx/CD = cd
Generic box classes: prefix with media-case media-box media-
  • Box = box
  • Box Wide = box-wd
  • Box Medium = box-md
  • Box Small = box-sm
  • Box Tall = box-tl
